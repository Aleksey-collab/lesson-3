# Списки.

Чтобы добавить ненумерованные списки, необходимо пункты выделить звёдочкой (*). Например вот так:
* элемент 1.
* Элемент 2.
* Элемент 3 .

Чтобы добавить нумерованные списки, необходимо пункты просто прономеровать. Например вот так:
1. первый элемент.
2. второй элемент.
3. третий элемент.
